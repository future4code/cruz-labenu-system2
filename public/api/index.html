Labenu System API Version 1.0

End Points

Class

  • GET /class list all classes
  • GET /class?name=cruz search class by name
  • POST /class create a new class
  • GET /class/name list class details
  • PUT /class/name update class
  • DELETE /class/name delete class
  • GET /class/students list all students in class
  • GET /class/teachers list all teachers in class
